# CtrlNode 
REST backend for sending commands

## Dependencies
Build Pistache: 

    git clone https://github.com/oktal/pistache.git
    mkdir build
    cd build
    cmake ..

## Building the library

    mkdir build
    cd build
    cmake .. -DPISTACHE_INC=<the pistache build lib dir> -DPISTACHE_LIB=<the pistache include dir>
